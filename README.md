# Hospital Queue Management System

## Project Description
This is a simple Hospital Queue Management System written in C language.  
It uses the Queue Data Structure and follows the FIFO (First In First Out) concept.

## Features
- Add patients
- Display patient queue
- Remove first patient
- Show remaining patients

## Technologies Used
- C Programming
- Queue Data Structure

## How to Run
1. Compile the program:
```bash
gcc hospital_queue.c -o hospital_queue
```

2. Run the program:
```bash
./hospital_queue
```

## Example Input
```text
Enter Number of Patients: 3
```

## Example Output
```text
Patient Removed:
ID: 101
Name: Rahul
```
